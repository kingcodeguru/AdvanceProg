### 2. Main Dashboard (My Drive)
The central hub for your files. Here you can navigate folders, view file grids or lists, and access the "Starred" and "Shared" tabs via the bottom navigation bar.
![mainpage](../images/mainpage.png)

---

### 3. File Actions
You can access the **File Menu** by clicking the three dots (`...`) on any file or folder. This allows you to:
* Rename
* Move
* Share
* Star/Unstar
* Delete

![file menu](../images/fileMenu.png)
